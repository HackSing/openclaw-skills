# 平台图片规格与推荐风格

> 生成前先查本文档确定尺寸和风格方向，再到具体风格文档中找 Prompt 模板。

---

## 平台规格速查

| 平台 | 比例 | size 参数 | 备注 |
|------|------|-----------|------|
| 推特封面 | 5:2 | `3200x1280` | 原生 5:2，无需裁切 |
| 公众号封面 | 2.35:1 | `3008x1280` | 原生 2.35:1，无需裁切 |
| 小红书 | 3:4 | `1728x2304` | 原生 3:4，无需裁切 |
| 产品图/Logo | 1:1 | `2048x2048` | 方形 |

> Seedream 4.5 支持 [1/16, 16] 范围内任意比例，只要总像素在 [3686400, 16777216] 区间即可。上述尺寸均已验证合规。

## 推荐风格调性

### 科技产品推广（当前主力风格）

- **背景**：深蓝/深紫渐变
- **色彩**：霓虹色（绿→黄→橙→红渐变表示层级）
- **元素**：扁平化图标、卡片式布局、代码框
- **字体**：粗体大标题 + 正文层次分明
- **适用**：开发者工具推广、开源项目宣传、技术方案对比
- **具体示例** → [text_card.md](text_card.md)

### 抽象氛围图（备选风格）

- **背景**：深色渐变
- **色彩**：高对比霓虹
- **元素**：几何形状、光效、仪表盘
- **特点**：不含精确文字，纯视觉传达概念
- **适用**：封面图、品牌视觉、不需要文字信息的配图
- **Prompt 语言**：英文优先（抽象描述时英文效果更好）

## 通用参数

所有平台生成时默认使用：

```
watermark: false
optimize_prompt_options.mode: standard
```
