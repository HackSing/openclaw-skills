# 精准文字信息卡 Prompt 示例

> 场景：`text_card` | 适用平台：小红书多图帖、知乎文章配图
> 核心技巧：用中文 Prompt + 极具体的文字内容描述，让 Seedream 直接渲染文字

---

## 关键经验

1. **用中文写 Prompt** — Seedream 对中文文字的渲染比英文 Prompt 描述中文内容更准
2. **逐行描述内容** — 不要笼统说"一个表格"，要把每一行的具体文字列出来
3. **颜色标签要明确** — 写"TRIVIAL绿色标签"而不是"第一个标签"
4. **开启提示词优化** — `optimize_prompt_options.mode: standard` 大幅提升质量
5. **原生 3:4 尺寸** — `size: 1728x2304`，无需后期裁切

```
一张科技风格的信息图海报，深蓝色渐变背景。

第1行："hi" / "你好"，硬规则，TRIVIAL绿色标签；
第2行："今天天气？"，0.16，SIMPLE浅绿标签；
第3行："对比React和Vue"，0.38，SIMPLE浅绿标签；
第4行："写AI趋势短文"，0.66，COMPLEX橙色标签；
第5行："写爬虫处理反爬"，0.59，COMPLEX橙色标签；
第6行："搭建消息队列"，硬规则，EXPERT红色标签。
底部文字"API费用直降60%"。
整体风格：现代科技感，深色背景，霓虹色彩，专业信息图。
```

**API 参数**：`size: 1728x2304, watermark: false, optimize_prompt_options.mode: standard`

## Prompt 编写模板

对于任何需要精确文字的信息图，按这个结构写：

```
一张[风格]的信息图海报，[背景色]渐变背景。
顶部大标题"[标题文字]"，白色粗体。
[副标题或说明，可选]
中间[布局方式：表格/卡片列表/上下对比]：
[逐行/逐块详细描述内容，包含精确文字]
底部[收尾文字或品牌信息]。
风格：现代科技，深色背景，专业信息图。
```
