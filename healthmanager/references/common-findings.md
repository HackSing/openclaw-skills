# HealthManager 常见风险项库

版本：2026-03-09

## 1. 插件读取环境变量并外发

### 常见判定
- 插件读取环境变量
- 同时存在网络发送逻辑
- 且没有足够清晰的边界说明

### 默认分级
- 通常判为 Critical

### 建议写法
- 检测到环境变量访问与网络发送组合
- 这类模式可能导致凭据或敏感上下文外发
- 需要立即人工审查源码和实际配置

## 2. gateway.trustedProxies 为空

### 常见判定
- 存在反向代理暴露场景
- 但未配置可信代理

### 默认分级
- Warn

### 建议写法
- 当前 trusted proxies 未配置
- 若通过反向代理暴露控制界面，相关头部可能被伪造

## 3. 工具权限或插件可达性偏宽

### 常见判定
- 某些插件工具默认可访问
- 缺少更细粒度的限制

### 默认分级
- Warn

### 建议写法
- 当前工具面偏宽
- 如无实际需求，建议收紧可用范围

## 4. 工作区 symlink 越界

### 常见判定
- workspace 内部通过 symlink 指向外部路径
- 存在目录边界不直观的问题

### 默认分级
- Warn

### 建议写法
- 当前存在 symlink 指向工作区外部目录
- 不一定是直接漏洞，但需要明确边界，避免误判和误写

## 5. 关键配置文件权限正常

### 常见判定
- 配置文件为 600
- 敏感目录为 700

### 默认分级
- Info

### 建议写法
- 当前关键配置文件权限正常
- 暂未看到明显过宽权限问题

## 6. 仅本地回环监听

### 常见判定
- 端口只绑定 127.0.0.1
- 无公网暴露

### 默认分级
- Info

### 建议写法
- 当前监听端口主要限于本地回环
- 暂未发现明显公网暴露面
